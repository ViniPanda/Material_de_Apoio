Área Administrativa

Para adicionar novos PDFs, siga os passos abaixo:

  1. Faça upload do arquivo PDF na pasta assets/pdfs.
  2. Abra o arquivo index.html no editor.
  3. Adicione uma nova linha na lista, no formato: 
    <li><a href="assets/pdfs/novo-arquivo.pdf" target="_blank">Nome do PDF</a></li>
  4. Salve as alterações e envie para o GitHub: 
    git add .
git commit -m "Adiciona novo PDF"
git push
